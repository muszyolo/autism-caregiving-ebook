About This GitHub Ebook

This HTML page is the live front door for the Autism Caregiving ebook. It is based on the source synthesis already prepared from the 21 provided journal articles on autism caregiving, family burden, resilience, service access, and caregiver-mediated intervention.

The goal of this version is simplicity. Instead of relying on Python code to regenerate the site, the GitHub Pages version is now directly editable as HTML, which makes maintenance easier and keeps the ebook site lightweight.

What’s Inside

  • A live GitHub Pages website in pure HTML.
  • A downloadable PDF version of the ebook.
  • A downloadable EPUB version for ebook readers.
  • A source manuscript preserved in the repository.

Main Themes

The ebook centers on caregiver journeys before diagnosis, family restructuring, emotional burden, coping and resilience, service gaps, and the role of caregiver-mediated intervention. It also keeps a source-notes structure so the synthesis remains visibly linked to the underlying articles.

The website is intentionally concise, while the downloadable ebook files retain the longer reading format.

Project Files

The key files for this HTML-based GitHub ebook are:

  • docs/index.html for the live website.
  • docs/autism_caregiving_ebook.pdf for the PDF download.
  • docs/autism_caregiving_ebook.epub for the EPUB download.
  • output/ebook/autism_caregiving_ebook.md for the manuscript text.
