# Autism Caregiving GitHub Ebook V2

Live page:
[https://muszyolo.github.io/autism-caregiving-ebook/](https://muszyolo.github.io/autism-caregiving-ebook/)

This project now uses a direct HTML page for the GitHub ebook site and presents version 2 of the ebook as a chapter-based HTML research synthesis.

## Main files

- `docs/index.html`: standalone HTML for the live GitHub Pages site and ebook v2
- `docs/autism_caregiving_ebook.pdf`: downloadable PDF ebook
- `docs/autism_caregiving_ebook.epub`: downloadable EPUB ebook
- `output/ebook/autism_caregiving_ebook.md`: manuscript source text

## Publishing

GitHub Pages serves the site from:

- branch: `master`
- folder: `/docs`

## Editing

To change the live page, edit:

```html
docs/index.html
```

No Python build step is required for the website anymore.

## Mapping

The HTML page now includes interactive world maps that visualize:

- study geography by paper count
- autism caregiving research themes by region

These maps are based on the papers in the ebook collection and show study coverage, not global autism prevalence.

## Version 2 improvements

Version 2 adds:

- clearer chapter-based ebook structure
- source matrix table for literature review use
- stronger thematic grouping
- recommendations for clinicians, policymakers, families, and researchers
- explicit limitations section
- improved reference presentation
